Welcome to Pakistan Real Estate Price Predictor

This application uses real-world real estate data of Pakistan (2023) to estimate property prices using a machine learning model. Whether you're buying, selling, or just curious about property trends, this tool can give you a realistic price prediction based on your input.

📊 About the Dataset

The dataset includes thousands of listings collected from various cities across Pakistan. Key features used to train the model include:

  • City – Location of the property (e.g., Lahore, Karachi, Islamabad)
  • Property Type – House, Plot, Flat
  • Purpose – For Sale or Rent
  • Area (in Marla) – The size of the property
  • Bedrooms & Bathrooms – Room and bath count
  • Price – The actual listed price

⚙️ Model Overview

The model applies feature engineering, log transformation, and linear regression to predict prices. It has been trained on cleaned and preprocessed data with R² score of approximately 0.82.
